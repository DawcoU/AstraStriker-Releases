# 🌌 AstraStriker 🚀

Witaj w **AstraStriker** – kosmicznej zręcznościówce! Poniżej znajdziesz krótką instrukcję, jak poprawnie uruchomić grę i jak nią sterować.

---

## 💻 Jak uruchomić grę? (Przeczytaj uważnie!)

Aby gra działała poprawnie i nie sypała błędami, zastosuj się do tych 3 kroków:

1. **WYPAKUJ ARCHIWUM ZIP!** Nie uruchamiaj gry bezpośrednio z wnętrza pobranego pliku `.zip`. Kliknij prawym przyciskiem myszy na pobrany plik i wybierz **Wypakuj tutaj** (lub użyj programu 7-Zip / WinRAR).
2. **Ignoruj ostrzeżenie Windows Defender (SmartScreen):**
   Gra nie posiada płatnego podpisu cyfrowego, dlatego Windows przy pierwszym uruchomieniu wyświetli niebieskie okienko ostrzegawcze. 
   * **Rozwiązanie:** Kliknij w tym okienku link **"Więcej informacji"**, a następnie przycisk **"Uruchom mimo to"**.
3. **Wymagana Java:**
   Do uruchomienia gry na komputerze musi być zainstalowana Java (JRE 17 lub nowsza). Jeśli jej nie masz, plik `.exe` wyświetli Ci powiadomienie z prośbą o jej pobranie.

---

## 🎮 Sterowanie w grze

* **Ruch statkiem:** `Statek będzie sam podążał za twoim kursorem myszy
* **Strzał:** `Spacja` lub `LPM` (Lewy Przycisk Myszy)
* **Pauza:** Klawisz `Esc`

---

## 📁 Zawartość folderu z grą

Dla poprawnego działania gry wszystkie poniższe pliki muszą znajdować się w tym samym, wypakowanym folderze:
* **`AstraStriker.exe`** – Główny plik uruchamiający grę.
* **`assets/`** – Folder zawierający wszystkie dźwięki, grafiki i tekstury (nie zmieniaj jego nazwy ani go nie usuwaj!).
